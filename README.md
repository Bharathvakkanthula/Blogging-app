# My Blog Platform

A customized blogging application built using Spring Boot and MySQL.

## Description
This project allows users to register, login, and create blog posts.
It follows MVC architecture and uses Spring Data JPA for database interaction.

## Features
- User authentication
- Create, update, delete blog posts
- Search posts by title
- Category management
- MySQL database integration

## Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA
- Thymeleaf
- MySQL
- Maven

## How to Run
1. Create MySQL database
2. Update application.properties
3. Run:
   mvn spring-boot:run
4. Open browser at http://localhost:8080
